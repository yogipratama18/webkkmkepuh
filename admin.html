Unggah Banyak Artikel dari Word

Tulis semua berita di satu file Word mengikuti format di bawah, lalu unggah di sini. Sistem akan otomatis membuatkan file data.js baru yang tinggal kamu ganti di folder situs.

Cara Pakai
1

Unduh template Word di bawah, lalu isi dengan berita kamu. Boleh satu file berisi banyak artikel sekaligus — pisahkan tiap artikel dengan baris ===.

2

Simpan file Word tersebut (.docx), lalu unggah di kotak bawah ini.

3

Periksa hasil pratinjau — pastikan tidak ada tanda merah. Kalau semua sudah benar, klik Unduh data.js.

4

Ganti file data.js lama di folder situs kamu dengan file yang baru diunduh, lalu upload ulang ke GitHub seperti biasa.

Format Tiap Artikel

Ketik persis seperti ini di Word, satu baris satu label. Bagian yang bertanda wajib harus diisi.

JUDUL: Kerja Bakti Bersihkan Saluran Drainase Lingkungan Karet   (wajib)
KATEGORI: Lingkungan                                              (wajib, lihat daftar di bawah)
LINGKUNGAN: Lingkungan Karet                                      (wajib)
TANGGAL: 20 Jul 2026                                              (wajib)
WAKTU BACA: 5 menit baca                                          (opsional)
FOTO: images/kerja-bakti.jpg                                      (opsional)
UTAMA: ya                                                         (opsional, jadi headline beranda)
ISI:
Paragraf pertama isi berita di sini.

Paragraf kedua, lanjutan cerita.

Paragraf ketiga, hasil atau dampaknya.
===

Kategori yang valid (ketik salah satu, huruf besar/kecil bebas): Kabar Desa, Pendidikan, Ekonomi & UMKM, Lingkungan, Kesehatan, Sosial
